About this tool


This tool has been made by A. Dodero and M. Sacchetti.
The goal of this tool is to show some informations about the road accidents analyzed during the BigData course assigned project.
This tool has been realized during the InfoVis course hold by Maurizio Patrignani (http://www.dia.uniroma3.it/~infovis) and uses the following technologies:

  • D3.JS
  • Bootstrap
  • MongoDB
  • Ajax
